== 공통 응답 포맷

모든 API는 `GlobalApiResponse` 형식으로 응답합니다:

[source,json]
----
{
  "success": true,
  "message": "요청에 성공했습니다.",
  "data": { ... }
}
----

[cols="1,1,2"]
|===
|필드|타입|설명

|success
|Boolean
|성공 여부 (true/false)

|message
|String
|응답 메시지

|data
|Object/Array/etc
|실제 응답 데이터 (API마다 다름)
|===
